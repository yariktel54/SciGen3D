3D Model Generator

About the Project

The 3D model generator offers several programs to create models based on different data types. Geo3D generates 3D models from black-and-white height maps (DEM) and colored height maps. Math3D creates models from mathematical formulas in the z=xy format. Space3D processes DEM and DTM from other planets, including colored maps. Uni3D is a universal program for generating models that supports jpg and png formats. Download the required program and start creating your own 3D models! You can also download or view ready-made models on the site for further exploration.

Instructions

  1. Prepare a height map or mathematical function file in a supported format (.png, .jpg).
  2. Download the program, enter coordinates, and click the "download tiles" button, or click "upload your tiles" and select the required map in the open window.
  3. Press the generate button. After the generation completes, an .stl file of the model will open.
  4. If desired, access additional generation settings by clicking the "advanced" button.
